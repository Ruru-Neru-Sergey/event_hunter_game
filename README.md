## 🚀 Быстрый старт

### 📋 Требования

- **Python** 3.8 или выше
- **Node.js** 16 или выше
- **npm** или **yarn** (для фронтенда)

### 💻 Установка

#### Способ 1: Автоматическая установка (рекомендуется)

```bash
# Клонируйте репозиторий
git clone https://github.com/yourusername/event-hunter-game.git
cd event-hunter-game

# Сделайте скрипт исполняемым
chmod +x setup.sh

# Запустите установку
./setup.sh
