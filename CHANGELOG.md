# Changelog

## [1.0.0] - 2024-03-23

### Added
- Initial release
- Real-time event prediction game
- AI-powered event predictions
- Leaderboard system
- Scoring system with confidence multipliers
- Auto-event generation
- WebSocket support
- React frontend with TypeScript
- Pure Python backend (no dependencies!)
- Docker support
- One-click setup script

### Features
- 3 event types: Goal, Strike, Foul
- 2 demo matches
- In-memory storage (Redis optional)
- ELO-like ranking system
- Responsive design with TailwindCSS
- Cross-platform support (Windows, Mac, Linux)

### Technical
- Backend: Python HTTP Server
- Frontend: React + TypeScript + Vite
- AI: Random Forest predictions
- Scoring: Time-based with multipliers
